# Data Dashboard

Dashboard interactivo de datos construido con React, TypeScript y Recharts.

## Tech Stack

- **React 18** + **TypeScript** + **Vite**
- **Recharts** — gráficas interactivas (área, barras, donut)
- **Tailwind CSS** — estilos responsivos
- **Lucide React** — iconografía
- Modo oscuro/claro

## Características

- KPIs con indicadores de cambio
- Gráfica de área — Ingresos vs Gastos (anual)
- Gráfica de barras — Crecimiento de usuarios
- Gráfica donut — Distribución por categoría
- Tabla de top productos con tendencias
- Sidebar de navegación
- 100% responsivo

## Instalación

```bash
npm install
npm run dev
```
